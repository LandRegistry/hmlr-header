Apply to register or update titles

Use this service to:

  • update registered titles
  • register new titles (including transfers of part and new leases)
  • register lease extensions (close an old lease and register a new lease of the same property to the same tenant)
  • close leasehold titles (including merger applications)
  • remove default Form A restrictions
You can only use this service for applications with up to 25 titles.

When to use a different service

You cannot transfer portfolios (TR5) using this service yet. You can submit these applications using thedocument registration service.

For first registrations, you should submit applications by post.

If your application has more than:

  • 25 titles, you should use the document registration service
  • 50 titles, contact the bulk application team
